# 📘 Hướng Dẫn Sử Dụng: PC Ultimate Optimizer v2.0

Chào mừng bạn đến với **PC Ultimate Optimizer** - công cụ tối ưu hóa hệ thống Windows mạnh mẽ, an toàn và dễ sử dụng.

---

## 📥 1. Cài đặt & Khởi động

### Yêu cầu hệ thống:
- **Hệ điều hành**: Windows 10 hoặc Windows 11 (64-bit).
- **Quyền hạn**: Quyền Quản trị viên (Administrator).
- **Internet**: Cần thiết cho tính năng cập nhật Driver và Phần mềm.

### Các bước thực hiện:
1. **Tải về**: Tải bản mới nhất từ [GitHub Releases](https://github.com/brillianfan/pcoptimizer/releases).
2. **Giải nén**: Giải nén file `.zip` vào một thư mục (Ví dụ: `C:\Tools\PCOptimizer`).
3. **Chạy ứng dụng**:
   - Double-click vào file **`PCOptimizer.bat`**.
   - Khi hộp thoại UAC hiện lên, chọn **Yes** để cấp quyền Admin.
4. **Bỏ qua cảnh báo Antivirus (nếu có)**: Do ứng dụng can thiệp vào Registry và hệ thống, một số phần mềm bảo mật có thể báo nhầm. Xem phần [An toàn](#shield-an-toàn--bảo-mật) bên dưới.

---

## 🚀 2. Các chức năng chính

### 📊 Dashboard (Bảng điều khiển)
- Xem nhanh thông tin CPU, RAM, Card đồ họa (GPU) và dung lượng ổ cứng.
- Theo dõi trạng thái hệ thống ngay khi mở app.

### 🧹 Deep Clean (Dọn rác chuyên sâu)
- **Dọn file tạm**: Xóa rác từ Temp, Prefetch, và Windows Cache.
- **Làm trống thùng rác**: Xóa vĩnh viễn các file trong Recycle Bin.
- **Xóa nhật ký hệ thống**: Làm sạch Windows Event Logs để giải phóng dung lượng.
- **Dọn shortcut lỗi**: Tìm và xóa các biểu tượng bị hỏng trên Desktop và Start Menu.
- **Windows Disk Cleanup**: Kích hoạt trình dọn dẹp mặc định của Windows ở chế độ nâng cao.

### 🔧 Tools & Optimization (Tối ưu hóa)
- **Internet Boost**: Tối ưu thiết lập mạng, giảm ping và tăng tốc độ duyệt web.
- **Optimize Registry**: Tinh chỉnh hệ thống để phản hồi nhanh hơn, giảm độ trễ menu.
- **Toggle Windows Update**: Bật hoặc tắt tạm thời cập nhật Windows khi cần thiết.
- **Disk Check**: Lên lịch kiểm tra và sửa lỗi ổ cứng (CHKDSK).

### 🗑️ Advanced Uninstaller (Gỡ ứng dụng)
- Liệt kê tất cả phần mềm đã cài đặt, bao gồm cả các ứng dụng tải từ **Microsoft Store**.
- Gỡ cài đặt tận gốc và quét dọn tàn dư trong `AppData` và `Registry`.

### 📦 Software Update (Cập nhật phần mềm)
- Sử dụng công nghệ Winget của Microsoft để quét và cập nhật các phần mềm phổ biến (Chrome, Zoom, Spotify...) lên bản mới nhất chỉ với 1 click.

### 🔌 Driver Update (Cập nhật Driver)
- Quét các Driver lỗi thời thông qua Windows Update API chính thống.
- Cập nhật Driver an toàn, không sử dụng nguồn bên thứ ba.

## 🛡️ 3. An toàn & Bảo mật

- **Mã nguồn mở 100%**: Bạn có thể kiểm tra từng dòng mã trên GitHub.
- **Không thu thập dữ liệu**: Ứng dụng không gửi bất kỳ thông tin cá nhân nào của bạn ra bên ngoài.
- **Sử dụng công cụ Microsoft**: Các tính năng dọn dẹp và cập nhật đều dựa trên các lệnh chuẩn của Windows (Winget, CHKDSK, cleanmgr).

### Tại sao Antivirus cảnh báo?
Ứng dụng sử dụng script để chỉnh sửa Registry và thực thi lệnh hệ thống - đây là hành vi mà các phần mềm Antivirus thường đánh dấu là "đáng ngờ" (False Positive). Bạn có thể yên tâm bỏ qua hoặc thêm ứng dụng vào danh sách ngoại lệ (Exclusion).
